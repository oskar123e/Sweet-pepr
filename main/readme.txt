installation
-------------------
option 1:
    run the following command;
    'sudo python3 setup.py install'

option 2:
    run the following command;
    'python3 inst.py'

difrence between the comands:
    option 1 installs it into the folder/directory the file is located
    option 2 installs it into a folder in your home folder named 'Sweet-pepr' also it downloads the latest version from github
